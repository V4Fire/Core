Interface RequestAPI

Object with API parameters. If the API is specified it will be concatenated with a request path URL. It can be useful to create request factories. In addition, you can provide a function as a key value, and it will be invoked.

You can provide a direct URL for the API, such as 'https://google.com'. Or you can provide a bunch of parameters to map on .api parameter from the application config. For example, if the config.api is equal to 'https://google.com', and you provide parameters, like,

  {
    domain3: 'foo',
    namespace: () => 'bar'
  }

then it builds a string is equal to 'https://foo.google.com/bar'.
