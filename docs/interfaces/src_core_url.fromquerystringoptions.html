Optional arraySyntax

arraySyntax: undefined | boolean
  • Defined in src/core/url/interface.ts:85

If true, then nested properties will be decoded by using [] syntax

default

false

example
// {foo: [1], bar: {bla: 2}}
fromQueryString('foo[]=1&bar[bla]=2', {arraySyntax: true});
