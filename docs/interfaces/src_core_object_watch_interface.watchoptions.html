Optional dependencies

dependencies: WatchDependencies
  • Defined in src/core/object/watch/interface.ts:170

List of dependencies for a path to watch. This parameter can help to watch getters.

example
const obj = {
  get foo() {
    return this.bla * this.baz;
  },

  bla: 2,
  baz: 3
};

watch(obj, 'foo', {dependencies: ['bla', 'baz']}, (val) => {
  console.log(val);
});

Optional tiedWith

tiedWith: undefined | object
  • Defined in src/core/object/watch/interface.ts:106

Link to an object that has connection with the watched object

example
const data = {
  foo: 2
};

class Bla {
  data = data;

  constructor() {
    watch(this.data, {tiedWith: this}, (val) => {
      console.log(val);
    });
  }
}

const bla = new Bla();
bla.foo = 3;
